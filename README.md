``IM_ESPORT_HUB``
Dette prosjektet er en nettside laget av oss. Nettsiden heter IM ESPORT HUB, og den er laget for at brukere skal kunne logge seg inn og melde seg på spilldager.

På linjen vår har vi faste spilledager, og gjennom denne nettsiden kan brukerne enkelt registrere seg og velge hvilket spill de vil delta i. Alt skjer gjennom innlogging og påmelding i systemet.

Målet med prosjektet er å gjøre det enklere å organisere spilledager, holde orden på hvem som deltar, og vise hvilken aktivitet som skjer hver spilledag.
